# LiveMark — Product Vision

## One-Liner

LiveMark is a fast, distraction-free Markdown editor where what you type is what you see — no split panes, no preview toggle, just writing.

## The Problem

Existing Markdown editors force users into one of two compromises:

1. **Split-pane editors** (VS Code, StackEdit) — you write raw Markdown on the left and see rendered output on the right. Your eyes constantly jump between two panels. The writing experience is fractured.

2. **Full WYSIWYG editors** (Google Docs, Notion) — they hide the underlying format entirely. You lose the precision and portability of Markdown. Formatting surprises are common.

3. **Typora** — the closest to ideal, but it's closed-source, its development pace has slowed, and its rendering can feel sluggish on large files. The UI carries legacy design choices.

Writers, developers, and students deserve an editor that renders Markdown **inline as they type**, feels **instant**, and stays **out of the way**.

## Vision Statement

**LiveMark makes Markdown invisible.** You type naturally. Headings become headings. Bold becomes bold. Links become clickable. You never leave your flow to "check the preview" — there is no preview. There is only your document.

## Target Users

| Segment | Need |
|---|---|
| **Software developers** | Write READMEs, docs, and design notes in Markdown daily |
| **Technical writers** | Need a focused tool for long-form Markdown content |
| **Students & researchers** | Take lecture notes and write papers in Markdown |
| **Bloggers & content creators** | Draft posts in Markdown for static-site generators |
| **Knowledge workers** | Use Markdown for personal notes and journals |

## Core Value Proposition

1. **Instant rendering** — Markdown syntax transforms as you type, not after you switch tabs
2. **Zero-config simplicity** — open, write, done. No plugins required for core functionality
3. **Native performance** — sub-100ms startup, smooth on 10,000+ line files
4. **True focus mode** — minimal chrome, maximum writing space
5. **Open and portable** — standard Markdown files, no vendor lock-in

## What LiveMark Is NOT

- Not a knowledge base (no graph view, backlinks, or database features)
- Not an IDE (no terminal, git integration, or language server)
- Not a collaboration tool (no real-time co-editing)
- Not a note-taking app (no tagging, organization, or search across files)

LiveMark does one thing extraordinarily well: **edit Markdown beautifully**.

## Long-Term Aspiration

Become the default Markdown editor on every developer's and writer's machine — the way that a good text editor is invisible infrastructure. Fast, reliable, always there.
